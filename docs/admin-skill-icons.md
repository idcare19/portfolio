# Skill Icon Names

Use these values in the admin `Skills` page inside the `Icon` field.

Supported icon names:

- `html`
- `css`
- `javascript`
- `bootstrap`
- `tailwind`
- `next`
- `react`
- `database`
- `mysql`
- `laravel`
- `server`
- `php`
- `appwrite`
- `wordpress`
- `git`
- `git-branch`
- `github`
- `testing`
- `check-circle`
- `deployment`
- `rocket`
- `vercel`
- `layout`
- `code`
- `wrench`

How to update:

1. Open Admin Dashboard -> Skills.
2. Edit the `Icon` field for any skill.
3. Paste one supported icon name from the list above.
4. Save Skills.

If the icon field is empty or unsupported, the UI falls back to the default code icon.
